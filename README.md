# 局域网多人 Excel 编辑器 v1.3.0

基于 Flask + WebSocket 的局域网多人实时协作 Excel 编辑工具。支持**按文件隔离的多人会话**和**多页签同时编辑**，打开同一文件的用户实时协作编辑，不同文件之间互不干扰。

**特点**：完全离线可用，无需互联网连接；支持多页签同时编辑多个文件；文件间可通过超链接跳转。

---

## ✨ 主要功能

### 🎯 核心特性
- **文件级会话隔离**：打开文件 A 的用户只能看到/编辑文件 A，与文件 B 的用户互不干扰
- **多人实时协作**：同一文件内的用户实时同步单元格修改
- **多页签编辑**：同时打开多个 Excel 文件，像浏览器标签一样切换
- **文件超链接**：在单元格中创建链接，点击跳转到其他文件
- **完全离线**：所有依赖已本地化，纯局域网运行
- **自动保存**：智能检测变更，仅在有变更时创建版本快照

### 📝 文件管理
- ✅ 文件夹分类管理（创建、重命名、删除）
- ✅ 新建空白 Excel 文件
- ✅ 上传本地 xlsx 文件
- ✅ 下载文件到本地
- ✅ 文件移动和重命名

### 🎨 样式编辑（v1.0.0 新增）
- ✅ **文字样式**：加粗、斜体、下划线
- ✅ **字体大小**：10px ~ 24px
- ✅ **文字颜色**：12 种常用颜色
- ✅ **背景颜色**：16 种常用颜色
- ✅ **对齐方式**：左对齐、居中、右对齐
- ✅ **样式持久化**：保存到 JSON 文件，随版本快照备份

### 🔗 多页签与超链接（v1.1.0 新增）
- ✅ **多页签编辑**：同时打开多个文件，页签栏切换（类似浏览器）
- ✅ **创建超链接**：选中单元格 → 点击"🔗 链接" → 选择目标文件
- ✅ **点击跳转**：点击蓝色超链接文字，在新页签打开目标文件
- ✅ **链接格式**：`LINK:目标路径|显示文本`
- ✅ **跨文件导航**：适合制作目录表、关联文档等场景

### 🔍 高级功能（v1.0.0 新增）
- ✅ **查找替换**（Ctrl+F）
- ✅ **撤销/重做**（Ctrl+Z / Ctrl+Y）
- ✅ **快捷键支持**（Ctrl+S 保存、Ctrl+B/I/U 样式）

### 👥 协作功能
- ✅ **实时显示其他用户**：彩色边框 + 用户名标签标识选中位置
- ✅ **独立颜色**：每个用户分配不同颜色（18 色循环）
- ✅ **平滑动画**：用户移动选中位置时平滑过渡

### ❄️ 冻结行列（v1.2.0 新增）
- ✅ **冻结行**：选中行后点击"❄️行"按钮，或手动输入行数
- ✅ **冻结列**：选中列后点击"❄️列"按钮，或手动输入列数
- ✅ **类似 Excel**：滚动时冻结区域保持可见
- ✅ **灵活设置**：支持同时冻结多行多列

### 🖌️ 格式刷（v1.2.0 新增）
- ✅ **复制格式**：选中源单元格 → 点击"🖌️"按钮 → 点击目标单元格
- ✅ **批量应用**：支持选中多个单元格批量应用格式
- ✅ **完整复制**：复制所有样式（粗体、颜色、对齐等）
- ✅ **实时同步**：格式刷应用后实时同步给协作用户

### 📊 单元格类型（v1.3.0 新增）
- ✅ **日期类型**：单元格显示日期选择器，支持 `YYYY-MM-DD` 格式
- ✅ **进度条类型**：输入 0-100 数值，显示可视化进度条
- ✅ **下拉列表类型**：配置选项列表，单元格变为下拉选择框
- ✅ **富文本类型**：单元格显示按钮，点击弹出富文本编辑器
- ✅ **文本类型**：默认类型，支持超链接和样式
- ✅ **批量设置**：选中多个单元格可同时设置类型
- ✅ **类型持久化**：保存到 JSON 文件，随版本快照备份
- ✅ **实时同步**：类型变更实时同步给所有协作用户

**使用方法**：
1. 选中一个或多个单元格
2. 从工具栏"单元格类型"下拉框选择类型
3. 下拉列表类型需要配置选项（每行一个）
4. 富文本类型点击"📝 编辑内容"按钮编辑

### 📝 富文本编辑器（v1.3.0 新增）
- ✅ **单元格显示为按钮**：简洁的按钮形式，节省空间
- ✅ **自定义按钮文字**：可设置按钮显示文字（如"📋 备注"、"📄 详情"）
- ✅ **弹窗编辑**：点击按钮打开富文本编辑器
- ✅ **完整格式支持**：标题、粗体、斜体、颜色、列表、对齐
- ✅ **插入链接和图片**：支持超链接和本地图片
- ✅ **实时预览**：所见即所得的编辑体验
- ✅ **内容同步**：富文本内容实时同步给协作用户

**富文本编辑器功能**：
- 三级标题样式（H1, H2, H3）
- 文字格式：粗体、斜体、下划线、删除线
- 文字颜色和背景色
- 有序列表和无序列表
- 文字对齐方式
- 插入超链接
- 插入图片

**使用方法**：
1. 选中单元格，设置类型为"📝 富文本"
2. 在弹出的配置框中输入自定义按钮文字（如"📋 备注"）
3. 单元格显示自定义文字的按钮
4. 点击按钮打开富文本编辑器
5. 编辑完成后点击"保存"
6. 内容自动保存并同步

**自定义按钮文字示例**：
- `📋 备注` - 用于备注信息
- `📄 详情` - 用于详细说明
- `🖼️ 图片` - 用于图片内容
- `📊 图表` - 用于数据图表
- `📝 编辑` - 简洁的文字提示

### 💾 版本管理
- ✅ **智能快照**：仅当内容变更时创建快照
- ✅ **最多 20 个版本**：自动清理最旧版本
- ✅ **一键回滚**：选择任意历史版本恢复
- ✅ **样式备份**：版本快照包含样式数据

### 📑 使用示例

**场景 1：多文件对比**
```
同时打开"预算表.xlsx"和"实际支出.xlsx"，方便对比数据
```

**场景 2：目录导航**
```
在"总览.xlsx"中：
A1: [2026年预算表] → 点击打开预算文件
A2: [项目进度表] → 点击打开进度文件
```

**场景 3：数据关联**
```
在"客户名单.xlsx"中点击客户名称，
自动跳转到该客户的"订单详情.xlsx"
```

**场景 4：冻结表头**
```
选中第1-2行，点击"❄️行"按钮，
滚动表格时表头始终可见
```

**场景 5：快速统一格式**
```
1. 选中已设置好格式的单元格（如红色加粗标题）
2. 点击"🖌️"格式刷按钮
3. 点击其他单元格，自动应用相同格式
```

**场景 6：设置单元格类型**
```
1. 选中项目进度列的单元格
2. 选择"单元格类型"→"📊 进度条"
3. 输入 0-100 的数值，自动显示进度条

或：
1. 选中状态列的单元格
2. 选择"单元格类型"→"📋 下拉列表"
3. 配置选项：待办/进行中/已完成
4. 单元格变为下拉选择框
```

**场景 7：使用富文本编辑器**
```
1. 在备注列选中单元格
2. 选择"单元格类型"→"📝 富文本"
3. 单元格显示"📝 添加内容"按钮
4. 点击按钮打开富文本编辑器
5. 输入详细说明，支持格式：
   - 标题、粗体、斜体
   - 列表、链接、图片
6. 点击保存，内容自动同步
```

**场景 8：制作项目看板**
```
在"项目看板.xlsx"中：
A列：任务名称（文本）
B列：负责人（下拉列表）
C列：进度（进度条类型，输入50显示50%）
D列：状态（下拉列表：待办/进行中/已完成）
E列：详细说明（富文本类型，可包含格式化文字和图片）
```

## 🚀 快速开始

### 方式 1：完整离线包（推荐）

1. 下载 `lan_excel_editor_offline_v1.3.0.zip`
2. 解压到任意目录
3. **安装 Python 3.9+**（勾选 "Add Python to PATH"）
4. 双击 `start.bat` 启动
5. 浏览器访问 `http://localhost:5000`

### 方式 2：源码运行

```bash
pip install -r requirements.txt
python app.py
```
