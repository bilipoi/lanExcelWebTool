# 局域网多人 Excel 编辑器

基于 Flask + WebSocket 的局域网多人实时协作 Excel 编辑工具。支持**按文件隔离的多人会话**，打开同一文件的用户实时协作编辑，不同文件之间互不干扰。

## 技术栈

| 组件 | 技术 |
|------|------|
| 后端 | Python / Flask / Flask-SocketIO / gevent |
| 实时通信 | WebSocket (Socket.io) |
| 会话隔离 | Socket.IO Room 机制 |
| Excel 读写 | openpyxl |
| 前端表格 | Handsontable |
| 前端 UI | 原生 HTML / CSS / JavaScript |

## 功能一览

### 按文件隔离的多人协作
- **文件级会话隔离**：打开文件 A 的用户只能看到/编辑文件 A，与文件 B 的用户互不干扰
- 同一文件内的多人实时协作：单元格修改实时同步给该文件的所有在线用户
- 顶部状态栏显示**当前文件**的在线用户列表
- 左侧文件树显示每个文件的在线人数徽章
- **离开文件**功能：可随时退出当前文件，回到未打开状态

### 文件管理
- **文件夹分类**：支持创建、重命名、删除文件夹，按目录分类管理文件
- **新建文件**：在任意文件夹下新建空白 xlsx 文件，自动打开并加入编辑
- **上传文件**：上传本地 xlsx 文件到服务器，上传后自动打开
- **下载文件**：将当前编辑的文件下载到本地
- **重命名**：右键文件或文件夹进行重命名
- **移动文件**：右键文件，选择目标文件夹进行移动
- **删除**：删除文件或文件夹（文件夹会连同内容一起删除）

### 版本快照与回滚（智能优化）
- 每次手动保存或自动保存时，**仅当内容有变更时**才创建版本快照
- 使用 MD5 哈希对比数据，避免生成无意义的重复快照
- 每个文件最多保留 **20 个历史版本**，超出后自动清理最旧版本
- 支持查看版本历史列表，选择任意版本进行恢复
- 恢复版本前会自动备份当前内容，防止误操作

### 权限控制
- 支持将文件设为 **只读** 或 **可编辑**
- 只读文件：顶部显示红色"只读"标记，表格锁定，无法编辑和保存
- 权限变更实时广播给该文件的所有在线用户，立即生效

### 基础样式编辑
支持对单元格应用基础样式，样式会实时同步给所有协作用户：

**文字样式**：
- 加粗 (B) / 斜体 (I) / 下划线 (U)
- 字体大小：10px ~ 24px
- 文字颜色：12 种常用颜色

**单元格样式**：
- 背景颜色：16 种常用颜色
- 对齐方式：左对齐 / 居中对齐 / 右对齐

**使用方法**：
1. 选中要设置样式的单元格（可批量选择）
2. 点击样式工具栏上的按钮应用样式
3. 样式会立即显示并同步给其他用户
4. 点击"清除格式"按钮可移除所有样式

**样式同步**：
- 所有样式变更实时同步到协作用户的界面
- 样式与单元格数据一起保存在会话中
- 重新打开文件时恢复之前的样式
- **实时显示其他用户的选中位置**：用带颜色的边框标识其他用户正在编辑的单元格
- **每个用户独立颜色**：系统自动分配 18 种不同颜色，便于区分不同用户
- **用户名标签**：在选中单元格上方显示彩色标签，标明当前操作者
- **平滑动画**：用户移动选中位置时，指示器平滑过渡到新位置
- **自动清理**：用户离开文件或断开连接时，自动移除其指示器
- **多 Sheet 支持**：仅在当前 Sheet 显示协作指示器，切换 Sheet 时自动更新

## 快速开始

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

依赖列表：
- Flask
- Flask-SocketIO
- openpyxl
- gevent
- gevent-websocket

### 2. 启动服务

```bash
cd lan_excel_editor
python app.py
```

启动后输出：
```
==================================================
  局域网多人 Excel 编辑器（按文件隔离）
  访问地址: http://0.0.0.0:5000
==================================================
```

### 3. 访问使用

- **本机**：浏览器打开 http://localhost:5000
- **局域网其他设备**：浏览器打开 http://你的IP地址:5000
  - Windows 查看本机 IP：在终端运行 `ipconfig`，找到 IPv4 地址

### 4. 使用流程

1. 打开页面后输入你的名字进入
2. 通过左侧文件树**点击文件**加入该文件的协作编辑
   - 文件树会显示每个文件的在线人数徽章 🔵
   - 选择已有徽章的文件可以快速加入协作
3. 在表格中直接编辑，修改会实时同步给**同一文件**的其他在线用户
4. 点击顶部"离开文件"按钮可以退出当前文件，回到未打开状态
5. 右键文件/文件夹可进行重命名、移动、删除、设置权限、查看版本历史等操作

### 5. 多人协作示例

**场景**：局域网内有 3 台电脑

| 用户 | 操作 | 效果 |
|------|------|------|
| 用户1（电脑A） | 点击"项目A/预算表.xlsx" | 进入该文件，开始编辑 |
| 用户2（电脑B） | 点击"项目A/预算表.xlsx" | **加入同一文件**，与用户1实时同步 |
| 用户3（电脑C） | 点击"项目A/进度表.xlsx" | **进入不同文件**，编辑互不干扰 |

用户1和用户2的修改会互相实时看到，用户3完全独立。

## 架构说明

### 按文件隔离的实现机制

1. **FileSession 类**：为每个打开的文件创建独立会话，包含：
   - 表格数据（内存缓存）
   - 在线用户列表
   - 上次保存的数据哈希（用于变更检测）
   - 只读权限状态

2. **Socket.IO Room**：每个文件是一个独立的 Room
   - 用户点击文件 → `join_room(file_path)`
   - 编辑事件 → `emit(..., room=file_path)`
   - 离开文件 → `leave_room(file_path)`

3. **自动清理**：当文件的所有用户都离开后，会话自动销毁，释放内存

### 智能快照机制

```python
# 伪代码示例
def auto_save():
    current_hash = md5(data)
    if current_hash == last_saved_hash:
        return  # 内容无变更，跳过保存和快照
    
    save_to_disk()      # 保存到 xlsx 文件
    create_snapshot()   # 创建版本快照
    last_saved_hash = current_hash  # 更新哈希
```

## WebSocket 事件

### 客户端 → 服务端

| 事件 | 参数 | 说明 |
|------|------|------|
| `join_file` | `{path, username}` | 加入指定文件的编辑房间 |
| `leave_file` | `{path}` | 离开当前文件 |
| `cell_edit` | `{sheet, row, col, value}` | 编辑单元格 |
| `cell_select` | `{sheet, row, col}` | 选中单元格（触发协作指示器） |
| `cell_deselect` | - | 取消选中单元格 |
| `save_file` | - | 手动保存当前文件 |

### 服务端 → 客户端

| 事件 | 数据 | 说明 |
|------|------|------|
| `file_opened` | `{path, filename, sheets, readonly, users, my_color, other_selections}` | 成功加入文件，接收初始数据 |
| `cell_updated` | `{sheet, row, col, value, username}` | 其他用户编辑了单元格 |
| `cell_selected` | `{sid, username, color, sheet, row, col}` | 其他用户选中了单元格（显示指示器） |
| `selection_cleared` | `{sid}` | 其他用户取消选中（移除指示器） |
| `user_joined` | `{username, color, users}` | 有新用户加入当前文件 |
| `user_left` | `{username, users}` | 有用户离开当前文件 |
| `file_readonly_changed` | `{path, readonly}` | 文件权限变更 |
| `file_saved` | `{by}` | 其他用户保存了文件 |
| `file_deleted` | `{path}` | 当前文件被删除 |
| `file_data_updated` | `{path, sheets}` | 文件内容被恢复（版本回滚） |
| `left_file` | - | 确认已离开文件 |
| `error` | `{message}` | 错误提示 |

## 注意事项

- 本工具仅支持 `.xlsx` 格式的 Excel 文件
- **按文件隔离**：打开不同文件的用户之间完全独立，无法互相看到或干扰
- 建议在局域网内使用，未做外网安全防护
- 默认端口为 5000，如需修改请编辑 `app.py` 最后一行的 `port` 参数
- 文件树中的在线人数徽章实时更新，方便选择协作对象
- 建议定期清理 `snapshots/` 目录，避免占用过多磁盘空间
