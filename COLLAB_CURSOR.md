# 协作指示器功能说明

## 功能概述

协作指示器功能让多人同时编辑 Excel 时，能够实时看到其他用户正在编辑哪个单元格，效果类似腾讯文档。

## 视觉效果

当其他用户选中单元格时，你会看到：

1. **彩色边框**：被选中的单元格显示 2px 粗的彩色边框
2. **用户名标签**：单元格上方显示彩色背景的用户名标签
3. **小圆点标记**：单元格左上角显示彩色小圆点

## 颜色分配

## 技术实现

### 后端 (app.py)

1. **颜色池**：`USER_COLORS` 列表定义 18 种颜色
2. **用户状态存储**：`FileSession.online_users` 存储每个用户的 `{username, color, selection}`
3. **选中事件处理**：
   - `cell_select`：用户选中单元格时广播给其他用户
   - `cell_deselect`：用户取消选中时通知移除指示器
4. **自动清理**：用户断开连接时自动移除其指示器

### 前端 (index.html)

1. **CSS 样式**：`.collab-cursor`、`.collab-cursor-border`、`.collab-cursor-label`
2. **全局状态**：`otherUsersSelections` 存储其他用户的选中位置
3. **事件监听**：
   - `afterSelection`：自己选中时发送 `cell_select` 事件
   - `afterDeselect`：取消选中时发送 `cell_deselect` 事件
   - `cell_selected`：收到其他用户选中，创建指示器
   - `selection_cleared`：收到其他用户取消选中，移除指示器
4. **位置更新**：表格滚动时自动更新指示器位置

## 使用场景

### 场景 1：查看谁在编辑

用户 A 正在编辑 B2 单元格，用户 B 的屏幕上会显示：
- B2 单元格有红色边框
- B2 单元格上方显示红色标签 "用户A"

### 场景 2：避免冲突

当用户 A 正在编辑某个单元格时，用户 B 可以看到该单元格被占用，从而选择编辑其他单元格，避免同时修改同一单元格。

### 场景 3：跟随协作

用户可以通过观察其他用户的选中位置，了解对方正在处理哪个区域，便于分工协作。

## 注意事项

1. **仅当前 Sheet**：协作指示器只显示在当前激活的 Sheet，切换 Sheet 后会根据其他用户的选中状态重新显示
2. **离开文件**：离开文件后所有协作指示器会清除
3. **性能优化**：最多支持 18 个用户同时协作（颜色池限制），超过后颜色会循环复用
4. **平滑过渡**：用户移动选中位置时，指示器会有 0.2 秒的平滑动画效果

## 测试方法

1. 在浏览器 A 中打开 http://localhost:5000，输入名字 "用户A"
2. 在浏览器 B 中打开 http://localhost:5000，输入名字 "用户B"
3. 两个浏览器都点击同一个文件
4. 在浏览器 A 中点击某个单元格
5. 在浏览器 B 中观察是否显示了带有 "用户A" 标签的彩色边框
