## 🎯 核心功能使用

### 📑 多页签编辑

**同时打开多个文件：**
1. 点击左侧文件树中的文件，会在新页签打开
2. 点击顶部页签栏切换不同文件
3. 点击页签上的 `×` 关闭文件
4. 支持同时编辑多个 Excel 文件

**使用场景：**
- 对比不同文件的数据
- 从一个文件复制数据到另一个文件
- 同时查看预算表和实际支出表

### 🔗 创建文件超链接

**步骤：**
1. 选中要添加链接的单元格
2. 点击样式工具栏的 "🔗 链接" 按钮
3. 从文件列表中选择目标文件
4. 可选：输入显示文本（默认为文件名）
5. 点击"创建链接"

**使用超链接：**
- 点击蓝色链接文字，在新页签打开目标文件
- 适合制作目录导航、关联文档等

**示例：**
```
在"总览.xlsx"中：
A1: [查看预算表] → 链接到"2024/预算.xlsx"
A2: [查看进度]  → 链接到"2024/进度.xlsx"
```

### ❄️ 冻结行列（v1.2.0 新增）

**冻结行：**
1. 选中要冻结的行（例如选中第 1、2 行）
2. 点击样式工具栏的 "❄️行" 按钮
3. 或使用对话框手动输入行数

**冻结列：**
1. 选中要冻结的列（例如选中 A、B 列）
2. 点击样式工具栏的 "❄️列" 按钮
3. 或使用对话框手动输入列数

**取消冻结：**
- 点击按钮后输入 0，即可取消冻结

**使用效果：**
- 滚动表格时，冻结的行/列保持可见
- 类似 Excel 的冻结窗格功能
- 适合固定表头或首列

### 🖌️ 格式刷（v1.2.0 新增）

**使用方法：**
1. 选中已设置好格式的**源单元格**
2. 点击样式工具栏的 "🖌️" 格式刷按钮
3. 按钮变橙色表示格式刷已激活
4. 点击**目标单元格**（或选中多个单元格）
5. 格式自动复制到目标单元格

**格式刷复制的内容：**
- 文字样式：粗体、斜体、下划线
- 字体大小和颜色
- 背景颜色
- 对齐方式

**取消格式刷：**
- 再次点击 "🖌️" 按钮
- 或按 Esc 键

### 📊 单元格类型（v1.3.0 新增）

支持为单元格设置特殊数据类型，提供更好的输入体验。

**支持的类型：**

| 类型 | 图标 | 说明 | 使用场景 |
|------|------|------|----------|
| 文本 | 📄 | 默认类型，支持超链接 | 普通文字内容 |
| 日期 | 📅 | 显示日期选择器 | 日期字段 |
| 进度条 | 📊 | 0-100 数值显示为进度条 | 项目进度、完成度 |
| 下拉列表 | 📋 | 从预定义选项中选择 | 状态、分类、选项 |
| 富文本 | 📝 | 显示为按钮，点击编辑 | 详细说明、备注 |

**设置单元格类型：**

1. **选中单元格**（可以选中多个批量设置）
2. 点击样式工具栏的 **"单元格类型"** 下拉框
3. 选择需要的类型

**配置下拉列表：**
1. 选中单元格
2. 选择类型 "📋 下拉列表"
3. 在弹出的对话框中输入选项（每行一个）
4. 点击确定

**使用进度条：**
1. 将单元格类型设为 "📊 进度条"
2. 输入 0-100 的数值
3. 单元格自动显示为彩色进度条

**清除类型：**
- 选中单元格后选择 "📄 文本" 类型即可恢复默认

### 📝 富文本编辑器（v1.3.0 新增）

富文本类型让单元格显示为简洁的按钮，点击后弹出完整的富文本编辑器。

**设置富文本单元格：**
1. 选中单元格
2. 选择类型 "📝 富文本"
3. 在弹出的配置框中设置按钮文字（如 "📋 备注"、"📄 详情"）
4. 单元格显示自定义文字的按钮

**配置按钮文字：**
- 可以使用任意文字或表情符号
- 常见用法：
  - `📋 备注` - 用于备注说明
  - `📄 详情` - 用于详细描述
  - `🖼️ 图片` - 用于图片展示
  - `📊 图表` - 用于数据图表
  - `📝 编辑` - 简洁提示

**编辑富文本内容：**
1. 点击单元格中的 "📝 添加内容" 或 "📝 编辑内容" 按钮
2. 在弹出的编辑器中输入内容
3. 支持的功能：
   - **标题**：H1、H2、H3 三级标题
   - **文字格式**：粗体、斜体、下划线、删除线
   - **颜色**：文字颜色和背景色
   - **列表**：有序列表和无序列表
   - **对齐**：左对齐、居中、右对齐
   - **链接**：插入超链接
   - **图片**：插入图片 URL
4. 点击 "保存" 保存内容

**使用场景：**
- 详细备注和说明
- 格式化的任务描述
- 包含图表的项目文档
- 富格式的评论和反馈

## 🚀 快速开始（离线环境）

### 方法一：完整离线包（推荐）

#### 第 1 步：在有网络的环境中准备

#### 第 1 步：准备依赖（在有网络的环境中）

1. 将项目复制到联网电脑
2. 下载 Python 依赖包：
   ```bash
   pip download -r requirements.txt -d packages
   ```
3. 等待下载完成，会创建 `packages/` 目录（包含所有 .whl 文件）

#### 第 2 步：打包分发

**方式 A：使用打包脚本（推荐）**
```bash
python create_offline_zip.py
```
生成 `lan_excel_editor_offline_v1.3.0_YYYYMMDD.zip` 文件

**方式 B：手动复制**
将整个项目目录（包括 `packages/`）复制到 U 盘或共享文件夹：

```
lan_excel_editor/          <-- 复制整个文件夹
├── packages/              <-- 包含所有 .whl 文件
├── static/
├── templates/
├── start.bat
├── requirements.txt
└── ...
```

#### 第 3 步：在离线服务器上部署

1. **安装 Python**（如果还没有）：
   - 下载 Python 3.9+ 安装包：https://www.python.org/downloads/
   - 安装时勾选 **"Add Python to PATH"**

2. **运行程序**：
   - 双击 `start.bat`
   - 首次运行会自动创建虚拟环境并安装依赖
   - 看到 "服务启动中..." 表示成功

3. **访问应用**：
   - 本机：http://localhost:5000
   - 局域网其他设备：http://[服务器IP]:5000
   - 查看服务器 IP：在 cmd 中运行 `ipconfig`

### 方法二：使用 pip 缓存（简化版）

如果目标机器可以临时联网，直接运行：

```bash
# 1. 安装依赖
pip install -r requirements.txt

# 2. 启动服务
python app.py
```

## 🔧 详细配置

### 修改端口号

编辑 `app.py`，找到最后一行：

```python
socketio.run(app, host='0.0.0.0', port=5000, debug=False)
```

将 `5000` 改为其他端口（如 `8080`）：

```python
socketio.run(app, host='0.0.0.0', port=8080, debug=False)
```

### 修改数据存储路径

编辑 `config.py`：

```python
# 数据目录（默认在项目目录下）
DATA_DIR = os.path.join(os.path.dirname(__file__), 'data', 'files')

# 可以改为其他路径，如：
# DATA_DIR = 'D:\\ExcelFiles'
```

### 防火墙设置

确保服务器防火墙允许访问端口 5000：

```powershell
# 以管理员身份运行 PowerShell
New-NetFirewallRule -DisplayName "Excel Editor" -Direction Inbound -Protocol TCP -LocalPort 5000 -Action Allow
```

## 🐛 常见问题

### 问题 1：提示 "未检测到 Python"

**解决**：
1. 访问 https://www.python.org/downloads/
2. 下载 Python 3.9+ 安装程序
3. 安装时勾选 **"Add Python to PATH"**
4. 重新打开命令行或重启电脑

### 问题 2：端口被占用

**解决**：
```bash
# 查看占用 5000 端口的进程
netstat -ano | findstr :5000

# 修改 app.py 使用其他端口（如 8080）
```

### 问题 3：局域网其他设备无法访问

**解决**：
1. 检查防火墙设置，允许端口 5000
2. 确认服务器 IP 正确：`ipconfig` 查看 IPv4 地址
3. 确保客户端和服务器在同一网段
4. 尝试关闭 Windows Defender 防火墙（仅测试时）

### 问题 4：依赖安装失败

**解决**：
1. 确保 `packages/` 目录存在且包含 .whl 文件
2. 运行 `download_packages.ps1` 重新下载
3. 或临时连接网络运行 `pip install -r requirements.txt`

### 问题 5：超链接点击没反应

**解决**：
1. 检查目标文件是否存在
2. 刷新页面后重试
3. 查看浏览器控制台（F12）是否有错误

### 问题 6：多页签切换后数据不更新

**解决**：
1. 这是正常行为，每个页签独立保存数据
2. 点击保存按钮（Ctrl+S）保存当前页签
3. 切换到其他页签查看最新数据

## 📁 数据备份

重要数据存储在以下位置：

- **Excel 文件**: `data/files/`
- **元数据**: `data/meta/`
- **版本快照**: `data/snapshots/`
- **样式文件**: `data/styles/`（自动隐藏）
- **类型文件**: `data/types/`（自动隐藏）

建议定期备份整个 `data/` 目录。

## 🔄 更新程序

1. 保留 `data/` 目录（包含所有文件和数据）
2. 替换其他文件为新版本
3. 重新运行 `start.bat`
